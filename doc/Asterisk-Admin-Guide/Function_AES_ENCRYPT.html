AES_ENCRYPT()

Synopsis

Encrypt a string with AES given a 16 character key.

Description

Returns an AES encrypted string encoded in base64.

Syntax

AES_ENCRYPT(key,string)
Arguments
  • key - AES Key
  • string - Input string
